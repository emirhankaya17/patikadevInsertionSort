
# Insertion Sort
`[22,27,16,2,18,6]`

### Yukarıda Verilen Dizinin Insertion Sort Aşamaları:
 ```
 1. [22|27,16,2,18,6]
 2. [22,27|16,2,18,6]
 3. [16,22,27|2,18,6]
 4. [2,16,22,27|18,6]
 5. [2,16,18,22,27|6]
 6. [2,6,16,18,22,27]
 ```
### Big-O:

O(n^2)

### Time Complexity:

* Average Case: O(n^2)
* Worst Case: O(n^2)
* Best Case: O(n)
